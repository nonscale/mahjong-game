# 🀄 효도 마작 (단아한 패 맞추기)

어머님의 치매 예방과 즐거움을 위해 만든 사천성 기반의 마작 패 맞추기 게임입니다.
조선 백자의 단아한 미학을 담아 눈이 편안하고 조작이 간편합니다.

## 🚀 배포 및 실행 방법

### 1. GitHub에 올려서 주소 만들기 (GitHub Pages)

전세계 어디서든 주소만 치면 접속할 수 있게 만드는 방법입니다.

1.  **GitHub 리포지토리 생성**: [GitHub](https://github.com/new)에 접속하여 새 저장소(예: `mahjong-game`)를 만듭니다.
2.  **파일 업로드**:
    *   `index.html`, `style.css`, `game.js`, `manifest.json`, `icon.png` 파일을 업로드합니다.
3.  **GitHub Pages 설정**:
    *   저장소의 **Settings** -> **Pages** 메뉴로 이동합니다.
    *   **Build and deployment** 섹션에서 Branch를 `main` (또는 `master`)으로 선택하고 **Save**를 누릅니다.
4.  **주소 확인**: 잠시 기다리면 `https://사용자아이디.github.io/mahjong-game/` 같은 주소가 생성됩니다.

### 2. 어머님 핸드폰에 '앱'으로 깔아드리기

생성된 주소를 어머님 스마트폰 브라우저(크롬 또는 사파리)에서 엽니다.

*   **안드로이드**: 주소창 옆 점 3개 메뉴 -> **'홈 화면에 추가'**
*   **아이폰**: 공유 버튼(화살표 모양) -> **'홈 화면에 추가'**

이제 핸드폰 홈 화면에 마작 패 아이콘이 생기며, 언제든 누르면 바로 게임이 실행됩니다.

## 🎨 디자인 철학: 검이블루 화이불치

*   **검소하되 누추하지 않고, 화려하되 사치스럽지 않다.**
*   어머님의 시력을 고려한 큼직한 패와 명확한 대비.
*   부드러운 애니메이션과 직관적인 힌트 기능.
